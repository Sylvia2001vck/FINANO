# FBTI设计

四维定义
R / S：稳健型 / 激进型（风险）
L / T：长线型 / 短线型（周期）
D / F：数据型 / 直觉型（决策）
C / A：集中型 / 分散型（仓位）
16 型 FBTI 精简介绍
稳健长线 RL
RLDC 持重者｜重仓长持，稳字当先
RLDA 均衡者｜均衡配置，稳中求进
RLFC 笃行者｜长期信仰，坚定持有
RLFA 温润者｜温和布局，从容获利
稳健短线 RT
RTDC 波段者｜数据择时，稳健波段
RTDA 套利者｜网格交易，稳健套利
RTFC 敏锐者｜轻仓博弈，盘感灵敏
RTFA 浅尝者｜小额试水，轻松理财
激进长线 SL
SLDC 掌舵者｜赛道龙头，重仓长持
SLDA 拓界者｜多线成长，均衡进攻
SLFC 孤勇者｜信仰重仓，专注赛道
SLFA 探路者｜布局未来，探索风口
激进短线 ST
STDC 破局者｜数据打板，快进快出
STDA 疾行者｜量化高频，纪律交易
STFC 逐风者｜追涨热点，狙击龙头
STFA 游牧者｜轮动热点，灵活试错
